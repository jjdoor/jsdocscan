# Changelog

## 0.1.0 — 2026-06-22

Initial release.

- Scan JS/TS/JSX/TSX/MJS/CJS files for exported functions and classes missing JSDoc
- Detect undocumented `@param` entries (warn-level)
- `--no-params` to skip param checks
- `--json` for machine-readable output
- `--quiet` for summary-only output
- `--ext` to customize file extensions
- Zero dependencies
